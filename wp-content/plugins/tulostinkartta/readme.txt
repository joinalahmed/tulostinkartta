Custom plugin part of the http://tulostimet.3d-tarvike.fi 3d printing site. 

Requires a child theme with various custom templates (Tulostinkartta Custom Community).

Used with several other WordPress plugins: 
    * 3DPrint Lite (Forked)
    * Advanced Custom Fields
    * BuddyPress
    * canvasio3D Light
    * GEO my WP
    
Uses the block.io API https://www.block.io/ for bitcoin payments.

This is a new and experimental version of the plugin and is not ready for use yet without heavy testing.

== Description ==

3D Printer owners can add their printers to a map after which they can be sent STL files for printing and they receive the payment via bitcoins.
